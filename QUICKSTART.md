### Core Concepts
- **Ink** = React for terminals
- **Yoga** = Flexbox layout engine
- **Box** = Layout container (like `<div>`)
- **flexDirection** = "row" or "column"
- **flexGrow** = How much space to take
- **padding** = Space inside
- **margin** = Space outside
